LLM-Powered Analytics Assistant with RAG
📌 Project Overview
A Streamlit-based assistant that enables business users to query structured e-commerce data and analyze unstructured customer reviews in plain English. It integrates NL-to-SQL for database queries, Retrieval-Augmented Generation (RAG) for review insights, and auto chart generation for visual analytics, all powered by GPT-3.5/4 and the Olist dataset.

The system integrates:

- NL-to-SQL for structured queries
- RAG (Retrieval-Augmented Generation) for review analysis
- Hybrid pipeline for combining both
- Auto chart generation with Plotly
- Streamlit UI for deployment

🛠️ Skills & Technologies:

- NLP, LLM Integration, RAG, Prompt Engineering
- Machine Learning, Deep Learning
- SQL, Data Visualization
- FAISS, Sentence Transformers
- SQLite, pandas, PyTorch
- OpenAI GPT-3.5/4 API
- Streamlit, Plotly
