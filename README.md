# LLM-Powered-Analytics-Assistant-with-RAG
A Streamlit-based assistant that enables business users to query structured e-commerce data and analyze unstructured customer reviews in plain English. It integrates NL-to-SQL for database queries, Retrieval-Augmented Generation (RAG) for review insights, and auto chart generation for visual analytics, all powered by GPT-3.5/4 and Olist dataset.
